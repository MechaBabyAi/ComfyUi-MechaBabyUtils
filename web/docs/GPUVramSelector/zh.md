# 按显存容量选择 (GPU VRAM Selector)

根据当前运行显卡的 **VRAM 容量**，从对应档位的输入中透传数据。支持任意类型，节点自动检测显存并输出匹配档位的输入。

## 输出

- **output**: 与当前显存档位匹配的输入值（透传）
- **vram_info**: 检测到的显存数值（如 `11.99 GB`）

## 显存档位（简化版）

| 输入 | 显存范围 | 典型显卡 |
|------|----------|----------|
| **<8G** | < 7.5 GB | GTX 1050 4G、1650 4G |
| **8G** | 7.5~9.5 GB | RTX 3050/4060/4070 8G |
| **10G** | 9.5~11.5 GB | RTX 3080 10G |
| **12G** | 11.5~14 GB | RTX 3060/3080 12G/4070 Ti |
| **16G** | 14~22 GB | RTX 4080/5080、20G 卡 |
| **24G** | 22~28 GB | RTX 3090/4090/5090 |
| **32G** | 28~40 GB | RTX 6000 Ada |
| **48G** | 44~56 GB | L40、A6000 48G |
| **80G** | 72~88 GB | A100 80G、H100 |
| **96G** | 90~102 GB | H100 96G、H200 |
| **>96G** | ≥ 102 GB | 大显存或多卡 |

## 匹配说明

实际显存数值（如 11.99 GB）会落在对应区间内匹配，例如 11.99G → 12G 档。
