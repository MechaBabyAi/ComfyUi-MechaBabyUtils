# 按显卡CC选择 (GPU CC Selector)

根据当前运行显卡的 **Compute Capability (CC)** 版本，从对应输入槽透传数据。支持任意类型，左侧连接不同上游节点，节点自动检测 GPU CC 并输出匹配的输入。

## 输出

- **output**: 与当前 GPU CC 匹配的输入值（透传，类型与连接的上游一致）
- **cc_version**: 当前检测到的 CC 版本字符串（如 `8.6`、`无 CUDA`）

## 输入说明（按显卡系列）

| 输入名称 | 对应显卡型号 | CC |
|----------|--------------|-----|
| **10系** | GTX 1050 / 1060 / 1070 / 1080 / 1080 Ti | 6.0, 6.1 |
| **16/20系** | GTX 1650 / 1650 Ti · RTX 2060 / 2070 / 2080 / 2080 Ti | 7.5 |
| **30系(CC8.0)** | A100, A30 等数据中心卡 | 8.0 |
| **30系(CC8.6)** | RTX 3050 / 3060 / 3070 / 3080 / 3090 / 3090 Ti | 8.6 |
| **40系** | RTX 4050 / 4060 / 4070 / 4080 / 4090 | 8.9 |
| **50系** | RTX 5050 / 5060 / 5070 / 5080 / 5090 | 12.0, 12.1 |
| **其它** | Maxwell、Volta、Jetson 等未单独列出的显卡 | 其它 CC |

## 使用场景

- 针对不同系列显卡使用不同模型或参数（如 INT4 仅 30 系及以上支持）
- 根据 CC 自动选择适配的 VAE、采样器或优化配置
